
# Project Title
  BLOGVERSE - A Userfriendly platform that allow users to READ , POST , EXPLORE blogs . 

## Tech Stack

**Client:** React , Javascript , CSS , HTML 

**Server:** Node, Express , MongoDB , JWT Tokens , Bcrypt
